A WebExtension to put tabs to sleep,
and automatically wake them up later.

Features

Nap Room

View all tabs & windows you have put to sleep.

 

Sleep Doctor

Mess with the sleep schedules for all tabs.

Keyboard Shortcuts

Too lazy to click? Set up your own key combinations.

 

Context Menus

Right click any link to snooze for later.

Dark Mode

Only the coolest kids use dark mode.

 

Notifications

Get Native notifications when tabs and windows wake up.

Highly Customisable

Who doesnt love settings?

 

Works Completely Offline

Your data will never be more secure.
